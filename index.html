Clase ocho:La estructura de una página web

  • <!DOCTYPE html>:Esta línea le dice al navegador que el documento es un archivo html.
  • <html>:Aquí es donde se colocan las etiquetas que no se ven en la página, como el título de la página y los enlaces a otros archivos.
  • <body>:Esta es la parte de la página que se muestra en el navegador. Aquí es donde colocamos todo lo que queremos que la gente vea en nuestra página web.

Clase nueve:Enlazar diferentes páginas

Para hacer que las personas puedan ir de una página a otra, usaremos la etiqueta code. <a> Dentro de esta etiqueta, escribimos el texto que queremos que se vea en el enlace y luego usamos el atributo href, para decirle al navegador a donde debe llevar a la persona cuando haga click en el enlace

Clase diez:Organizando con la etiqueta div.

La etiqueta div Es como una caja en la que puedes poner cualquier cosa que quieres. Sirve para organizar y estructurar tu página web. Puede poner varios elementos dentro de un div para agruparlos y darles estilo.

¡Esto es un Div!

En este div puedes poner texto, imágenes, videos o cualquier otro elemento que desees.

Colocar el video

Para poner un video en nuestra página web, usaremos la etiqueta <video>. Dentro de esta etiqueta especificamos la ruta del archivo de video, utilizando el atributo src Tambien agregamos el atributo controls para que aparezcan los controles de reproducción del video y la persona pueda reproducirlos, pausarlo y ajustar el volumen
